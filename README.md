# 👧 Analyse des Prénoms Féminins Américains (1880-1889)

## 📌 Contexte du Projet
Ce projet analyse les données des prénoms féminins aux États-Unis sur la période **1880-1889**. 
L'objectif est d'identifier les tendances, la popularité, la croissance et la concentration 
des prénoms dans cette décennie charnière.

---

## 🎯 Objectifs
- Identifier le **Top 5 des prénoms féminins** les plus populaires.
- Analyser l'**évolution annuelle** de ces prénoms.
- Calculer le **Taux de Croissance Annuel Moyen (TCAM/CAGR)**.
- Mesurer la **volatilité** (écart-type) des prénoms.
- Évaluer la **concentration du marché** via l'indice HHI.

---

## 🛠️ Technologies Utilisées
- **PostgreSQL**
- **SQL** (CTEs, sous-requêtes, fonctions statistiques)
